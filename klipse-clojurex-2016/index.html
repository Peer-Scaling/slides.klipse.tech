How to write an interactive README for a clojure[script] project

  • Go to your repo settings and configure gh-pages on your docs folder
  • Create docs/index.html
  • Use the page source of Conway's game of life as a template
  • Configure the klipse snippets with data attributes as explained in details here: config options.
  • Visit the KLIPSE repository to stay up to date with latest developments and discover other cool webpages with interactive code snippets.
